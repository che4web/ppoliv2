# AquaGreen (Gin + GORM + Bootstrap)

Демо-проект главной страницы интернет-магазина систем автополива.

## Стек
- Go 1.22
- Gin (HTTP + роутинг)
- GORM + SQLite (хранение данных страницы)
- html/template (рендер через Gin)
- Bootstrap 5.3 (CDN)

## Запуск
```bash
go mod tidy
go run ./cmd/web
```

Сайт будет доступен по адресу: `http://localhost:8080`.

При первом запуске создаётся `app.db`, выполняются миграции и сидирование стартовых данных.
